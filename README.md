# 0000.0120
Indicator boards for 1U enclosures

Small boards for 5, 6, 10 or 12 LED indicators with a maximum height of 36mm for usee in 1U rack mounting enclosures. These are for Kingbright L-93A8CB and L-93A8EB PCB mounting indicators, and attach to a main board using 0.1" pitch right-angled pin headers.
